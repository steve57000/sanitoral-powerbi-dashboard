# Sanitoral - Tableau de bord Power BI

Projet de data visualisation réalisé dans le cadre du parcours Data Analyst.

Objectif : créer un tableau de bord dynamique permettant de suivre l’avancement des projets IT et Marketing de Sanitoral, d’identifier les écarts de performance et d’aider les directeurs à prendre des décisions.

## Contenu du repository

- `mission/` : fichiers originaux fournis pour la mission.
- `livrables/` : fichiers produits et rendus finaux.
- `ressources/` : éléments de travail complémentaires.
- `notes/` : suivi, décisions et journal de bord.

## Outils utilisés

- Power BI Desktop
- Power Query
- DAX
- Excel
- Git / GitHub
