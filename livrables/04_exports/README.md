# Exports du dashboard

Les captures du dashboard doivent etre ajoutees manuellement dans :

```text
livrables/04_exports/captures_dashboard/
```

Le PDF exporte doit etre ajoute manuellement, si demande, dans :

```text
livrables/04_exports/export_pdf/
```

Ne pas creer automatiquement de capture, de PDF ou d'autre fichier binaire depuis l'assistant.
